### `thumnail()` 함수를 만들어서 메인 섹션 네 개에 적용하셈 ㅇㅇ
```javascript
function thumnail() {
  // <img> element
  // <h1> element : title
  // <div> element : dot x 2
  // <p> element : paragraph x 2
}
```
