## 필기내용
### 원시코드 vs. 목적코드
- 원시코드 -> 개발용 -> 동적인 방식 -> devdependency
- -> 컴파일 -> 
- 목적코드 -> 표시용 -> 정적인 방식 -> dependency
### babel
- 리액트는 바벨을 의존한다
- JSX를 바벨이 자바스크립트로 만들어준다
### npm vs. npx
- npm(node package manager) -> 매니저, 관리용, 설치용, 제어판
- npx(node package execute) -> 클릭, 실행
### 라이브러리 vs. 프레임워크
- 리액트는 라이브러리
- CRA는 프레임워크
