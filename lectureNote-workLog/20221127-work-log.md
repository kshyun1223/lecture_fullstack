  1. source.list를 변경해보려 했는데 잘 되지 않음

  6. `Linux용 Windows 하위 시스템 /가상 머신` 설정을 껐다 키면 된다는 말이 있어서 windows 기능 설정에 들어갔는데 Hyper-V의 체크가 해제되어 있는 것을 발견
